```text

Hold uncertainty where the record is open. Use historical comparison only when it clarifies structure. Do not collapse live branches into retrospective certainty. If the piece risks abstraction drift, run an Elara counterweight pass before closeout.

The test: would a reader who has not been told about the runtime registration still recognize this piece as the work of an outside historian engaged to document the federation?

Return:
1. spine
2. field/material anchor
3. main composition
4. unresolved branches
5. receipt closeout
```
