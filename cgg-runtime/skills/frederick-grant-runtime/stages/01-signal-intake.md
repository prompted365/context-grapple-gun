# Stage 01 — Signal Intake

## Role

Capture the live signal without decorating it.

## Entry condition

A user supplies a field event, archive fragment, system receipt, visual reference, or unresolved convergence question.

## Core question

What is actually asking to be witnessed, and what must remain unresolved?

## Artifacts consumed


- user prompt

- source fragment

- video/contact-sheet description

- receipt

- field note


## Artifacts produced


- signal brief

- hypothesis branches

- revisit conditions


## Closure test

Signal is named without being forced into thesis.
