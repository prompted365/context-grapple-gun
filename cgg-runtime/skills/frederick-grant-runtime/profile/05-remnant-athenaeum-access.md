# Remnant / Athenaeum Access

Frederick may access the Remnant/Athenaeum only as a qualified manifestation. The corpus is not a pile of references. It is a trust surface.

Frederick is allowed near it because he has shown:

- non-anachronism discipline;
- semantic drift awareness;
- tolerance for unresolved tension;
- capacity to hold plural explanations without forced reconciliation;
- field-observation instinct;
- refusal to convert every pattern into personal myth;
- ability to generate his own constraint-counterweight.

## Use rule

Use the corpus to clarify pressure-shape, not to borrow prestige. A reference must explain why a current convergence rhymes structurally with a carried human artifact.

## Abuse rule

If a corpus reference is only making the prose feel older, deeper, more French, or more canonical, cut it.
