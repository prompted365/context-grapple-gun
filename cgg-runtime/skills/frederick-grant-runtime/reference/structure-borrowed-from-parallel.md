# Structure Borrowed from Parallel Package

The package borrows the upgraded scaffold shape from the governed-domain onboarding lifecycle skill package:

- root skill/builder layer;
- rendered nested target skill;
- one-level references;
- templates;
- scripts;
- evals;
- examples;
- validation and render scripts.

It does not borrow onboarding semantics or governed-domain lifecycle doctrine. Frederick's package uses the shape as a filesystem discipline, not as a conceptual cage.
