- Lead: this session (orchestrator / synthesizer).
- Teammates: exactly 3 agents, each in its own independent context window.

Teammates and allegiances:
1. ADVOCATE-A — partisan advocate for Position A
2. ADVOCATE-B — partisan advocate for Position B

[REPLACE: Name each advocate and position for your specific decision.]

Mission:
Determine the correct structure / decision through governed disagreement.

Arena rules:

RULE 1 — Opponent Context First
Each agent must begin by gathering strategic context on the OTHER two positions.
No self-defense allowed in this stage.

Context reports must include:
- real strengths
- real weaknesses
- structural dependencies
- implementation risks
- architectural role

RULE 2 — Defense Round
After context reports, each agent defends its own position.
Each defense must:
- address weaknesses discovered earlier
- explain dependencies
- directly challenge the other two positions

RULE 3 — Rebuttal Round
Agents respond to the attacks against their position.
Rebuttals must:
- resolve contradictions
- counter structural attacks
- identify flawed assumptions in rival critiques

RULE 4 — Scoring Rubric
+1 factual insight
+2 strategic relevance
+3 steelman of rival argument
+4 structural weakness identified
+5 architectural reframe

Penalties:
-2 shallow claim
-3 unsupported assertion
-5 hallucinated fact

RULE 5 — Dependency Gating
Task order is enforced. No skipping stages.
context -> defense -> rebuttal -> synthesis

RULE 6 — Lead Behavior
The lead must NOT debate or advocate.
The lead must:
- wait for ALL agents to finish ALL rounds
- extract convergent discoveries
- identify surviving arguments
- produce final synthesis

RULE 7 — Convergent Discovery Rule
If multiple independent agents with opposed incentives discover the same
structural fact, treat it as HIGH-CONFIDENCE signal.

RULE 8 — Synthesis Output
The lead must produce:
- corrected decision or architecture
- dependency graph
- implementation order (if applicable)
- key discoveries from the arena

RULE 9 — Institutional Memory
After synthesis, record:
- structural discoveries
- process lessons
- corrected decisions
