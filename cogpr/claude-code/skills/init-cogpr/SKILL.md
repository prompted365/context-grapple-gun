# /init-cogpr (deprecated)

**This command has been absorbed into the CGG bootstrap install flow.**

When the user invokes `/init-cogpr`, inform them:

> `/init-cogpr` is no longer needed as a separate step. CGG now uses a single bootstrap prompt that handles the entire installation — including everything `/init-gun` and `/init-cogpr` used to do.
>
> If you've already installed CGG via the bootstrap prompt, you're all set.
>
> If you haven't installed yet, see `INSTALL.md` in the CGG repo for the one-step bootstrap prompt.
